# 软件工程课程简介

## 课程评价

无尽的画图，厚重的砖头。

理了一遍 PPT ，发现 PPT 的结构就是依托，软件工程最大、最核心、最本质的要求，我认为是通俗易懂，不论是项目代码、文档、算法。然而软件工程的教学往往不是这样，无尽的、乱七八糟的、通常他们自己也不太清楚的概念，会被僵硬的塞到正在低头玩手机的学生耳中，在期末时让学生们回味无穷。
