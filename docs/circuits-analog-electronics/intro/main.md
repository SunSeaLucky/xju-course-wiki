# 电路与模电

## 课程评价
入门电路的基础课程，奈何教考的内容比较基础，没有详细的讲解。

并且期末考试的答案离谱，误差计算结果1.227W Multisim仿真结果1.23W 被扣1分

## 考点与注意
如果想严谨地每一步都计算结果，建议使用卡西欧计算器
